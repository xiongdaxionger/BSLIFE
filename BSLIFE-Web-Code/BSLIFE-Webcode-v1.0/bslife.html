BSLIFE为国内以人为中心的垂直新零售品牌，创立于2018年，总部位于广东省深圳市， 隶属于百思狼文化科技有限公司。通过线上+线下两种结合形式，服务于城市以20-45岁 年轻工薪阶级，提供全天24小时服务。

BSLIFE的坚持为城市年轻工薪阶级提供“有科学,有品质,有美感的产品, 及生活基本服务" 的及美好购物体验，期望在新时代成为设计为用户提供更多价值。

2018年, BSLIFE,深圳南山店正式开业,以一种全新的形式，向全国范围内扩张，正式开启 “BSLIFE新零售“。目前，店面也开启,品牌形象以崭新的风格呈现给深圳的年轻消费者, 并将以更好,更快,更有价值的方式转递到全国各大城市.
